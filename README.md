Building
--------
To build this repository, first clone it and create a build folder.
```bash
$ git clone https://github.com/anazli/tic-tac-toe.git
$ cd tic-tac-toe
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Running
--------
```bash
$ bin/tictactoe
```
